## Lessons Learned
### [2026-04-30]
- **Optimantra RESULT_ONLY 標準範式**：sftp_result_path=/Prod/Input/（vendor 共用）, sftp_archive_path=空字串, sftp_ordering_path=null, report_option=PERSONALIZED, kit_delivery_option=NO_DELIVERY, ordering_enabled=0/result_enabled=1，不需 order_clients/sftp_folder_mapping

- **Provider ID missing 的標準處理流程**：(a) 先檢查 ticket description（PM 可能後來補）; (b) DB 反查 customer_name 看是否能找到; (c) 都沒有 → Jira comment 問 PM
